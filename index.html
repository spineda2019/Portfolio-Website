Major Qualifying Project:

Aerial Image Recognition Using Machine Learning


In this project, in conjunction with four other students, our faculty advisor, and our sponsoring organization (The U.S. Army Combat Capabilities Development Command), we are working on improving on an iteration of this project performed the last academic year, which was able to develop a parachute drop simulator. In this year's iteration, our team worked towards making the simulation portable and easy to use for end-users, as well as improving the object recognition in the machine learning models.
